# AVTO 3 Game

Мини игра три-в-ряд для Telegram mini-app, ориентирована на мобильный запуск в Vercel.

## Запуск локально
- Поставь любой статикус (`npm i -g serve`) и выполни `serve .` или открой `index.html` через `npx http-server`.
- Все ассеты лежат в корне и в `items/`, скрипты — в `src/`, стили — `styles/main.css`.

## Сборка кода
- Вся логика на ES-модулях без сборщика. Таймер, уровни и перетаскивание живут в OOP-классах.
- Ключи GitHub/Telegram держи в `.env` (шаблон — `.env.example`) и не коммить.

## Деплой в Vercel
- Тип проекта: Static. Output Directory: корень.
- Убедись, что `background.png`, `box.png`, `box_front.png`, `shelf.png`, папка `items/` и `index.html` публикуются.

## Структура
- `src/engine` — игровые сервисы (таймер, борд, генератор предметов, контроллер перетаскивания).
- `src/models` — сущности коробок, полок и предметов.
- `styles/main.css` — компактная мобильная верстка.
